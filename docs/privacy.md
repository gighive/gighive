# GigHive Privacy Policy

GigHive is designed so that media files (audio, video) and user accounts are stored on the user’s own self-hosted server.  
GigHive does not collect, store, or share personal information.  

GigHive gathers the [**bare minimum of information for debugging purposes**](TELEMETRY_ENDUSER.md) during installation. If you do not want GigHive to collect this minimal installation telemetry, there are opt-outs in both installation flows.

**Contact:** contactus@gighive.app for any privacy-related questions.
